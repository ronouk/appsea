##### Main functionalities:

1. Search, in real time
2. Sort by name and size (asc and desc)
3. Data presentation in graph form
4. Loader, alert and errorpage

##### Package used:

1. React router
2. Daisy UI
3. TailwindCSS
4. Toastify
5. Sweet Alert 2
6. React Loader Spinner
7. React Icons
8. Rechart
